# Примеры использования MaixPy MCP Server

После настройки сервера в VS Code, вы можете использовать GitHub Copilot Chat для работы с документацией MaixPy v4.

## Базовые запросы

### Поиск информации о модуле

```
@workspace Как использовать модуль camera в MaixPy?
```

Copilot автоматически найдет и предоставит документацию по модулю camera.

### Получение API документации

```
@workspace Покажи API документацию для модуля image
```

### Список доступных модулей

```
@workspace Какие модули доступны в MaixPy v4?
```

## Советы по эффективному использованию

1. **Используйте @workspace** - это говорит Copilot использовать контекст вашего workspace, включая MCP серверы

2. **Будьте конкретны** - чем точнее вопрос, тем лучше ответ:
   - ❌ "Расскажи про камеру"
   - ✅ "Как настроить разрешение 1920x1080 для камеры в MaixPy?"

3. **Запрашивайте примеры кода** - Copilot может генерировать код на основе документации:
   ```
   @workspace Напиши код MaixPy для детекции лиц с выводом на дисплей
   ```

4. **Используйте итеративный подход** - начните с общего вопроса, затем уточняйте:
   ```
   @workspace Расскажи про модуль nn в MaixPy
   > [получаете обзор]
   
   @workspace Покажи пример загрузки модели YOLOv5
   > [получаете конкретный пример]
   ```

5. **Проверяйте актуальность** - всегда уточняйте версию:
   ```
   @workspace Это актуально для MaixPy v4?
   ```

## Проверка работы MCP сервера

Чтобы убедиться, что сервер работает:

1. Откройте VS Code
2. Откройте GitHub Copilot Chat
3. Спросите:
   ```
   @workspace Список всех модулей MaixPy v4
   ```

Если Copilot отвечает со списком модулей (camera, image, display, nn и т.д.), сервер работает корректно!

## Отладка

Если что-то не работает:

1. **Проверьте логи MCP:**
   - View → Output
   - Выберите "GitHub Copilot Chat" в выпадающем списке
   - Ищите сообщения от maixpy-docs сервера

2. **Тестовый запуск:**
   ```powershell
   python mcp_server\server.py
   ```
   (Сервер будет ждать ввода, используйте Ctrl+C для выхода)

3. **Проверка Python:**
   ```powershell
   python --version
   pip list | findstr mcp
   ```
