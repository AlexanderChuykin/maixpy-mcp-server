   pip install -r requirements.txt

2. 📝 Настройка VS Code
   Добавить в settings.json:
   {
     "github.copilot.chat.mcp.servers": {
       "maixpy-docs": {
         "command": "python",
         "args": ["c:\\dev\\maixpy_mcp\\mcp_server\\server.py"]
       }
     }
   }

3. 🔄 Перезапуск VS Code
   Полностью закрыть и открыть VS Code

Сервер не работает?

1. Проверьте установку:
   pip show mcp

2. Запустите тесты:
   python tests\test_server.py

3. Проверьте логи VS Code:
   View → Output → GitHub Copilot Chat

4. Проверьте путь в settings.json
