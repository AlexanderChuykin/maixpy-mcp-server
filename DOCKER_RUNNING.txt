📋 СЛЕДУЮЩИЙ ШАГ: Настройка VS Code

1. Откройте файл настроек VS Code:
   
   Windows: %APPDATA%\Code\User\settings.json
   
   Или через VS Code:
   - Ctrl+, (открыть настройки)
   - Нажмите иконку файла справа сверху

2. Добавьте эту конфигурацию:

   {
     "github.copilot.chat.mcp.servers": {
       "maixpy-docs": {
         "url": "http://localhost:8000/sse"
       }
     }
   }

3. Сохраните файл (Ctrl+S)

4. Полностью перезапустите VS Code:
   - Закройте все окна VS Code
   - Откройте VS Code заново

💡 СОВЕТЫ

1. Контейнер автоматически перезапускается при перезагрузке системы

2. Документация монтирована как volume - обновления применяются сразу:
   • Измените файлы в ./doc_md/
   • docker compose restart
   • Пересборка НЕ требуется!

3. Если меняете код сервера:
   • docker compose down
   • docker compose build
   • docker compose up -d

4. Для просмотра внутренностей контейнера:
   • docker exec -it maixpy-mcp-server bash
